Class: RequestClient

RequestClient

new RequestClient()

A specialized esri.workers.WorkerClient that sends XHR requests through a specific worker. Functional replacement for dojo/_base/xhr

Methods

<static> getClient(callback, name) → {esri.workers.RequestClient}

Creates and returns a esri.workers.RequestClient with a callback set in the worker from the passed module path. Calling this function without any arguments returns a RequestClient instance.

Parameters:
Name Type Argument Description
callback string <optional>

either a require path to a script file containing the callback function (See WorkerClient.addWorkerCallback) for more information.
name string <optional>

the function name in the callback script to use as the callback function. @default "main"
